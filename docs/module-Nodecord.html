Module: Nodecord

Functions for generating objects that provide access to the API. For each secret key a new object should be created. The objects returned by connect() will each maintain their own connection to the gateway. Multiple objects should not be created with the same key.

Methods

(inner) connect(token, isBot)

Return an object that will create a connection to the gateway. This provides all the functionality of the object returned by register(), but also maintains it's connection allows the use of methods that require the gateway or to have 'identified'.
Parameters:
Name Type Default Description
token string The authorization token to use for the requests.
isBot boolean true Is the client a bot.
